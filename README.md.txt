# 4-Bit ALU in Verilog

This project implements a simple 4-bit ALU (Arithmetic Logic Unit) capable of performing:

- Addition
- Subtraction
- AND
- OR
- XOR
- Equality Check

## Files

- `alu4bit.v` - Main ALU module
- `alu4bit_tb.v` - Testbench to verify functionality

## How to Simulate

Use tools like:
- EDA Playground (Icarus Verilog)
- ModelSim
- Vivado Simulator
